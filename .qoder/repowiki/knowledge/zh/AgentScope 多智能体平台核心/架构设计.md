- **分层解耦设计**：采用 `model`（模型）、`tool`（工具）、`agent`（逻辑）与 `app`（服务）的分层架构，通过 `formatter` 模块实现不同 LLM 厂商协议的标准化转换。
- **服务化与 UI 协同**：内置基于 FastAPI 的 `app` 模块提供多租户 Agent 服务，并通过 `examples/web_ui` 中的前后端分离应用实现可视化交互与状态管理。
- **环境隔离机制**：通过 `workspace` 模块抽象本地、Docker 及 E2B 运行环境，确保 Agent 在执行 Bash 等高风险工具时的安全性与资源隔离。
- **依赖按需加载**：利用 `pyproject.toml` 定义的可选依赖组（如 `service`, `workspace`），配合代码层面的懒加载策略，实现核心库的轻量化与功能扩展的灵活性。
